Ajouter Kjio CSS

Rien de bien compliqué, il vous suffit simplement d'inclure les 2 fichiers suivants.

Surcharges SCSS

Si vous passez par les fichiers SCSS pour construire et modifier le framework, un fichiers vous permettra de surcharger les variables existantes. Il s'aggit du fichier _surcharges-vars.scss dans le dossier assets/scss.

Sections

En ajoutant la classe .section à une balise section, vous lui ajoutez un padding-top et un padding-bottom de 60 pixels.

Containers

Le container vous permet de centrer vos blocs. Il faut simplement ajouter la classe .container à un bloc.

En ajoutant la classe .is-fluid, le container prend 95% de la largeur du viewport.

La grille

Modals

Tout d'abord, il vous faut créer un bouton avec les attributs data-action="modal" et data-target="ID de votre modal".

Puis il faut créer la modal (de préférence en bas de document). Il existe 3 types de modals : .is-full, .is-medium et .is-small
